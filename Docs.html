Object-Oriented Programming

Copy Constructors

A Copy constructor is an overloaded constructor used to declare and initialize an object from another object.By Default, Complier cretes a Default constructor and a copy constructor.But if we create any contructor then complier does't create one.Copy constructor serves many purposes like when we want to convert one data type to other. In the C++ programming language, a copy constructor is a special constructor for creating a new object as a copy of an existing object. Copy constructors are the standard way of copying objects in C++, as opposed to cloning, and have C++-specific nuances. The first argument of such a constructor is a reference to an object of the same type as is being constructed, which might be followed by parameters of any type.

#include<iostream>
using namespace std;
class A
{
int a,b;
public:
A(int x, int y) //copy constructor
{a = x; b = y;}
A(A &x)
{ a = x.a; b = x.b; }
void display()
{cout<<"a:"<<a<<" b:<<"b;}
};
int main()
{
A a1(3,4);
A a2(a1);
a1.display();
a2.display();
return 0;
}

MultiLevel Inheritance

Inheritance is one of the important principle of object-oriented programming.In Inheritance, the data member as well as the methods of Base class are inherited in derived class.Private data members are not accessible in derived class.In MultiLevel Inheritance,data members of class A are inherited by class B and data members of class B are inherited by class C. If a class is derived from another derived class then it is called multilevel inheritance. So in C++ multilevel inheritance, a class has more than one parent class.

#include<iostream>
using namespace std;
class A
{
public:
int a;
A()
{ a = 10; }
};
class B:public A
{
public:
int b;
B()
{ b = 5;}
int get()
{ return a;}
void sum()
{ cout<<a+b;}
};
class C: public B
{
public:
int c;
C()
{ c = 20;}
};

int main()
{
B b1;
C c1;
b1.sum();
cout<<"\n"<<c1.a<<" "<<c1.b<<" "<<c1.c<<endl;
return 0;
}

Function Overloading

C++ provides the right to have different functions with same name.Those functions although have same name but differ in number of arguments or the type of arguments.Like in a program to find sum of different data types, we can name those similar to improve the clearity of code. Function overloading is a feature in C++ where two or more functions can have the same name but different parameters. When a function name is overloaded with different jobs it is called Function Overloading. In Function Overloading “Function” name should be the same and the arguments should be different.

#include<iostream>
using namespace std;
class A
{
public:
void sum(int x,int y) //overloaded function
{ cout<<"Sum is "<<x+y;}
void sum(double i,double j) //overloaded function
{ cout<<"Sum is:"<<i+j;}
void sum(double i,double j,double k) //overloaded function
{ cout<<"Sum is:"<<i+j+k;}
};
int main()
{
A a1;
a1.sum(3,4); //this will call function with two int type arguments
a1.sum(12.5,2.5); //this will call function with two double type arguments
a1.sum(11.5,2.5,1.5); //this will call function with three double type arguments
return 0;
}

Operator Overloading

In C++,we create a new data type with either structure or classes.But only creating data type is not sufficient, we also need to define some opertions to be performed on those data types.Here operator overloading plays an important role.Opertors are overload to perform same opertion on user-defined data types. In C++, we can make operators to work for user defined classes. This means C++ has the ability to provide the operators with a special meaning for a data type, this ability is known as operator overloading. For example, we can overload an operator ‘+’ in a class like String so that we can concatenate two strings by just using +.

#include<iostream>
using namespace std;
class A
{
int a,b;
public:
A(){ a = b = 0;}
A(int x,int y)
{a = x; b = y;}
void display()
{cout<<"a:"<<a<<" b:"<<b;}
A operator+ (A x) //operator overloading
{
A temp;
temp.a = a + x.a;
temp.b = b + x.b;
return(temp);
}
};
int main()
{
A a1(3,4),a2(5,6);
A a3;
a3 = a1+a2;
a3.display();
return 0;
}
